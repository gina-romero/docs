_Definitions of the different participation states you'll see in Great Question studies._

### Overview

Participant statuses help you track where individuals are in the research process. These states appear throughout studies to make it easier to see who has been invited, scheduled, or completed a session.

#### Participation States

| **Status** | **Definition** |
| --- | --- |
| **Shortlisted** | The candidate has been selected as a participant for participation. |
| **Applied** | The participant has applied to participate in the study. |
| **Invited** | An invitation has been sent to the participant. |
| **Scheduled** | The participant has been scheduled into a confirmed time slot. |
| **Started** | The participant has started your test or survey. |
| **Completed** | The participant's participation has been completed |
| **No Show** | The participant did not attend their scheduled session. |
| **Removed** | The participant was removed from participating. |
| **Canceled** | The participant's scheduled participation was canceled. |
| **Opted out** | The participant has opted out of participating in that particular study. |
